# Sharp-Style (シャープ書式)
`#`とその他いろいろな記号で、日本語の仮名文字を表記する書式です。

ローマ字からまぁまぁ影響を受けています。

簡易的なplayground -> https://likedloki.github.io/sharp-style/

## <概要>
まずは実物から。

`#'2, #"1, #%6` で `みかん` になる。

`#5, #:"4, #%6, #"2, #:$4, #!3, #"1, #/|` で `おげんきですか。` になる。

よく見てもらうと分かりますが、おおよそ仮名文字と一対一で対応しています。
一文字の区切りは `,(カンマ)` もしくは ` (半角スペース)` です。

実際の記法について説明します。説明に当たって使う表現は以下の通りで...

- `[1-5]` = 1から5までの整数。
- `[1-5 + ?6]` = 1から5までの整数。場合によっては6も加える。
- `[A?]` = Aはつけても付けなくてもいい。
- `[A...]` = Aの個数は問わない。

---
### 1. 母音型
`#[1-5]` で表され、「あ、い、う、え、お」を表記できます。
- `あ` = `#1`
- `い` = `#2`
- `う` = `#3`
- `え` = `#4`
- `お` = `#5`

と言ったぐあい。

---
### 2. 子音型
`#[子音記号][1-5 + ?6]`で表され、五十音の大半を表記できます。

子音記号というのは、五十音における**行**を表します。
- `"`(ダブルクオート) = **か行**
- `!`(感嘆符) = **さ行**
- `$`(ドル記号) = **た行**
- `%`(パーセント) = **な行**
- `&`(アンパサンド) = **は行**
- `'`(シングルクオート) = **ま行**
- `(`(括弧開き) = **や行**
- `)`(括弧閉じ) = **ら行**
- `=`(イコール) = **わ行**

子音記号をローマ字における子音[k("), s(!), t($)...]、後ろの数字をローマ字における母音[a(1), i(2), u(3), e(4), o(5)]として見ればわかりやすいと思います。

- `#"1, #"3, #"5` = `ka, ku, ko` = `かくこ`
- `#!1, #!2, #'2` = `sa, si, mi` = `さしみ`

ほぼすべて`[1-5]`の数字しか使えませんが、`%`(な行)だけ例外として`6`まで使えます。これは`ん`を表すためであり、`#%6` = `ん` となります。

また、`や行`と`わ行`については、五十音表に基づくので
- `#(1` = `や`
- `#(2` = `い`
- `#(3` = `ゆ`
- `#(4` = `え`
- `#(5` = `よ`

とか

- `#=1` = `わ`
- `#=2` = `ゐ`
- `#=3` = `う`
- `#=4` = `ゑ`
- `#=5` = `を`

になります。

---
### 3. 装飾型
`#[装飾記号][子音記号][1-5 + ?6]`で表され、濁音や半濁音、拗音を表記できます。

装飾記号というのは以下の通りで、子音記号の前にくっつけます。

- `:`(コロン) = **濁点**
- `;`(セミコロン) = **半濁点**
- `_`(アンダーバー) = **拗音**

上三つに加え、特殊枠として↓
- `^`(キャレット) = **通常文字**(現時点で使う必要はない)

などがあります。
- `#"1, #:"1, #'2` = `ka, ga, mi` = `かがみ`
- `#!4, #%6, #'1, #2` = `ze, xn, ma, i` = `ぜんまい`
- `#"2, #_(1, #"3` = `ki, xya, ku` = `きゃく`
- `#1, #_#1` = `a, xa` = `あぁ`(**あ行**の場合は子音記号に`#`を使う)

---
### 4. 長音型
`#--` `#-~` `#~-` `#~~` のパターンで書くことができ、`ー`や`～`を表記できます。

表記↓
- `#--` = `ー`
- `#-~` = `ー`
- `#~-` = `～`
- `#~~` = `～`

長音型に関しては、同じ文字に対して記法が二つありますが、基本的にどっちでもいいです。

---
### 5. 句読点型
`#/-` `#/|` のパターンで書くことができ、名の通り読点と句点を表記できます。

表記↓
- `#/-` = `、`(読点)
- `#/|` = `。`(句点)


---
## <コードの使い方>
まずnpmでインストールします。 (npmの基本操作を知ってる前提)
```
npm install sharp-style
```
下記のように使う
```javascript
import shrp from "sharp-style"

const resultA = shrp.stringify("りんごはおいしいとおもうよ");

// "#)2, #%6, #:"5, #&1, #5, #2, #!2, #2, #$5, #5, #'5, #3, #(5"
console.log(resultA.string);

// ['#)2', '#%6', '#:"5','#&1', '#5', '#2','#!2', '#2', '#$5', '#5',  "#'5", '#3', '#(5']
console.log(resultA.array);

const resultB = shrp.parse("#%4, #'3, #2, #%1, #_#1");

// "ねむいなぁ"
console.log(resultB.string);

// ["ね", "む", "い", "な", "ぁ"]
console.log(resultB.array);
```


## <その他>
既に妄想の中でいくつかの省略形を思いついているんですが、時間がないので、とりあえず基本記法だけコードで変換できる状態です。これについてはいずれ作ります。気になる人向けに、以下にその省略記法について書きます。

---
### 1. 母音省略
`#[装飾記号?][子音記号][1-5][子音記号...]`で表され、直前の母音を暗黙的に引き継ぎ、母音数字を省略することができます。

具体例としては
- `#%1', #"5` = `#%1, #'1, #"5` = `na, ma, ko` = `なまこ`
- `#&1:", #%4` = `#&1, #:"1, #%4` = `ha, ga, ne` = `はがね`

---
### 2. 子音省略
`#[装飾記号?][子音記号][1-5...]`で表され、直前の子音記号を暗黙的に引き継ぎ、子音記号を省略することができます。

具体例としては
- `#%12345` = `#%1, #%2, #%3, #%4, #%5` = `na, ni, nu, ne, no` = `なにぬねの`
- `#"23` = `#"2, #"3` = `ki, ku` = `きく`

 ---
### 3. 拗音における省略
`#[装飾記号?][子音記号][1-5]_[子音記号][1-5 + ?6]`で表され、拗音を書く際に`#`を省略できます。

具体例としては
- `#"2_(1` = `#"2, #_(1` = `きゃ`
- `#$3_$3, #"5, #'2` = `#$3, #_$3, #"5, #'2` = `つっこみ`
 
また、拗音のあとに省略形として続けて書くこともでき、例えば、拗音の子音か母音を引き継ぎたいなら、`^`(キャレット)を使います。
- `#(1_#1^":", #'2` = `#(1, #_#1, #"1, #:"1, #'2` = `やぁかがみ`

---
### 4. 母音シフト
`#[装飾記号?][子音記号][1-5][[シフト記号][子音記号]...]`で表され、五十音表に基づき、母音を直前の段から上げ下げしたうえで、次の子音記号に当てはめます。

シフト記号とは

- `<`(アップシフト) = **直前の母音より一つ分上げる**
- `>`(ダウンシフト) = **直前の母音より一つ分下げる**

具体例としては
- `#"1<` = `#"1, #2` = `かい`
- `#"3>"` = `#"3, #"2`  `くき`

子音記号に`#`があるか、そもそも何も書いてない場合は、**あ行**がそのまま適応される。
連続して書くこともできるが、以下のようになる。
- `#1<<<<` = `#1, #2, #3, #4, #5` = `あいうえお`
- `#"1<<!` = `#"1, #2, #!3` = `かいす`
